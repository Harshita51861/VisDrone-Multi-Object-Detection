# VisDrone Multi Object Detection

This project implements AI-based multi-object detection using the VisDrone dataset.

## Technologies Used
- Python
- OpenCV
- Deep Learning
- Computer Vision

## Dataset
VisDrone Dataset:
http://aiskyeye.com

## Features
- Object detection from aerial images
- Bounding box visualization
- Model evaluation

## Output
The model detects objects such as cars, pedestrians, bicycles, and buses from drone images.
